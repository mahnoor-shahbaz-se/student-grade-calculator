# Student Grade Calculator (Java)

A simple Java console application that calculates student grades based on marks entered by the user.

## Overview
This project was built to practice programming fundamentals, logical thinking, and basic data handling using Java.

The application takes marks as input, calculates total marks, percentage, and assigns a grade based on defined criteria.

## Features
• Input marks for multiple subjects  
• Total marks calculation  
• Percentage calculation  
• Grade assignment based on performance  
• Input validation  
• Simple console-based interface  
• Beginner-friendly structure  

## Concepts Used
Java fundamentals  
Control structures (if/else, switch)  
Loops  
Functions and modular programming  
User input handling  
Basic arithmetic operations  

## How to Run the Project
1. Download or clone the repository  
2. Open the project in IntelliJ IDEA  
3. Run the Java file (`StudentGradeCalculator.java`)  
4. Enter marks when prompted in the console  
5. View calculated grade and percentage
